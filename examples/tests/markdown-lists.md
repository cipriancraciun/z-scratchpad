## markdown lists
## -- format: commonmark


## tight (one level)

* item 1
* item 2
* item 3

## tight (two levels)

* item 1
  * item 1-a;
  * item 1-b;
* item 2
  * item 2-a;
  * item 2-b;
* item 3
  * item 3-a;
  * item 3-b;


## loose (one level)

* item 1

* item 2

* item 3

  * item 1-b;

* item 2:

  * item 2-a;
  * item 2-b;

* item 3:

  * item 3-a;

  * item 3-b;
